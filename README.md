Arabic words cloud
================

This python script generates word cloud from arabic texts. Read more about word cloud.
To avoid reverted text issue, I used [/mpcabd/python-arabic-reshaper].

## Required packages

Use pip to install :

* numpy
* pandas
* matplotlib
* pillow
* wordcloud
